# RHDH Blueprints Documentation

Welcome to the Red Hat Developer Hub (RHDH) Blueprints documentation.

## Overview

RHDH Blueprints provide pre-configured templates and patterns for quickly scaffolding projects, services, and infrastructure within the Red Hat Developer Hub ecosystem.

## Quick Links

- [Getting Started](docs/getting-started.md) - Set up and start using blueprints
- [Blueprints Overview](docs/blueprints/overview.md) - Learn about available blueprints
- [Creating Blueprints](docs/blueprints/creating.md) - Build your own blueprints
- [Configuration Reference](docs/reference/configuration.md) - Configuration options

## Features

- **Pre-built Templates**: Ready-to-use project scaffolds
- **Customizable**: Extend and modify blueprints to fit your needs
- **Integrated**: Works seamlessly with RHDH and Backstage

## Getting Help

If you encounter issues or have questions:

1. Check the [Configuration Reference](docs/reference/configuration.md)
2. Review existing [blueprints](docs/blueprints/overview.md) for examples
3. Open an issue in the repository
