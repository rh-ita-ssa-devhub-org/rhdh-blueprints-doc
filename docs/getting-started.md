# Getting Started

This guide will help you get up and running with RHDH Blueprints.

## Prerequisites

Before you begin, ensure you have:

- Access to Red Hat Developer Hub
- Appropriate permissions to create and manage blueprints
- Git client installed locally

## Installation

### Step 1: Access the Developer Hub

Navigate to your RHDH instance and log in with your credentials.

### Step 2: Browse Available Blueprints

1. Go to the **Create** section in the sidebar
2. Browse the available software templates
3. Select a blueprint that matches your needs

### Step 3: Configure Your Blueprint

Fill in the required parameters:

| Parameter | Description | Required |
|-----------|-------------|----------|
| `name` | Project name | Yes |
| `description` | Brief description | Yes |
| `owner` | Team or user owner | Yes |
| `repository` | Target repository location | Yes |

### Step 4: Create Your Project

Click **Create** and wait for the scaffolding process to complete.
